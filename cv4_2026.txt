Moderní počítačová grafika

Animace
•

Časovače
• vyvolání událostí bez interakce uživatele
• dvě základní formy
• Timer – událost se provede po uplynutí definovaného časového intervalu

• registrace: glutTimerFunc(unsigned msecs, void∗ func, int value)
• msecs – interval, za který se funkce volá znovu
• func – callback (logika co se má dít při opětovném volání)
• value – celé číslo, které se prodá callback funkci (např. ID Timeru, externí stav)

• Idle – událost se vyvolá vždy, když má aplikace přidělen od OS strojový čas a nezpracovává žádnou

jinou událost, (nestabilní, radši se vyhnout)

• registrace: glutIdleFunc(void∗ func)

Double buffering
•

Pokud provedení funkce glFlush() trvá déle než jeden vertikální překreslovací takt
obrazovky, scéna se v průběhu vykreslování liší mezi jednotlivými po sobě následujícími
snímky.

•

Projevuje se to mizejícími a objevujícími se objekty na scéně.

•

Zapnutí double bufferingu pomocí funkce
• glutInitDisplayMode(unsigned int mode)
• Módy:

•

•

GLUT_RGBA true-color vykreslovací režim

•

GLUT_SINGLE bez double bufferingu

•

GLUT_DOUBLE vytvoří další framebuffer a zapne double buffering

•

GLUT_DEPTH vytvoří tzv. depth-buffer (nebo také z-buffer)

Přepnutí bufferů:
• glutSwapBuffers()
